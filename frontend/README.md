# Frontend (Angular + Material)

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run dev server:
   ```bash
   npm start
   ```
3. Open `http://localhost:4200`

## Environment

Edit `src/app/environments/environment.ts`:

- `apiBaseUrl`: FastAPI base URL (default: `http://localhost:8000/api/v1`)
